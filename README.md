# Idle Test

## Project Description
This project is a simulation game that allows players to experience an idle gameplay environment. Users can engage with various interactive elements that progressively develop the game state without the need for constant active engagement.

## How to Run the Game
1. Clone the repository:
   ```bash
   git clone https://github.com/mtzj91/idle-test.git
   cd idle-test
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the game:
   ```bash
   npm start
   ```
4. Open your web browser and navigate to `http://localhost:3000` to play.

## Features Overview
- **Engaging Gameplay**: Features mechanics that allow players to progress idly.
- **Interactive Elements**: Players can interact with various elements to enhance their experience.
- **Progress Tracking**: Players can track their progress in the game and see how their actions are affecting the game state.
